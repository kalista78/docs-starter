## What Every Token Gets

Regardless of how you create it, every Slinq token includes:

| Feature | Details |
|---------|---------|
| **Fixed Supply** | 100 billion tokens (no inflation) |
| **Locked Liquidity** | LP locked permanentlyâ€”no rugs possible |
| **Creator Rewards** | 40% of all trading fees go to you |
| **MEV Protection** | Dynamic fees protect against bots at launch |
| **PancakeSwap Listing** | Instant trading on BNB Chain's largest DEX |

## Comparison

| Feature | Twitter | Farcaster | Web |
|---------|---------|-----------|-----|
| Speed | Instant | Instant | ~30 seconds |
| Custom Image | No | No | Yes |
| Description | Auto-generated | Auto-generated | Custom |
| Social Links | No | No | Yes |
| FDV Selection | Default (500 BNB) | Default (500 BNB) | 50-50,000 BNB |
| Wallet Needed | No | No | Yes |

## Which Should You Use?

<Tabs>
  <Tab title="Choose Twitter if..." icon="x-twitter">
    - You want maximum speed
    - Your Twitter followers are your target audience
    - You don't need custom branding
    - You want the simplest experience
  </Tab>
  <Tab title="Choose Farcaster if..." icon="message">
    - You're active in the Farcaster ecosystem
    - You want to reach crypto-native users
    - Speed is your priority
    - You're already in Warpcast
  </Tab>
  <Tab title="Choose Web if..." icon="browser">
    - You want full control over your token's appearance
    - You need custom images and descriptions
    - You want to set a specific starting price
    - You're building a brand around your token
  </Tab>
</Tabs>

## Rate Limits

To maintain platform quality, each user has daily creation limits:

| Platform | Tokens Per Day |
|----------|---------------|
| Twitter | 2 |
| Farcaster | 4 |
| Web | 3 |

<Info>
Rate limits reset at midnight UTC. If you need higher limits for a legitimate project, contact us.
</Info>
